# Movie Recommendation System

This project was developed as part of the DecodeLabs Artificial Intelligence Internship Program.

## Project Goal

Create a simple recommendation system that suggests movies based on user interests and preferences.

## Features

* User preference input
* Similarity-based matching
* Movie recommendations
* Similarity score display
* Modern web interface
* Responsive design

## Technologies Used

* Python
* Flask
* HTML
* CSS

## Available Genres

* Action
* Comedy
* Sci-Fi
* Horror
* Drama

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open:

http://127.0.0.1:5000
